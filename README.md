# GTA IV 100% Companion PWA

A mobile-first, offline-capable GTA IV completion checklist.

## Notes

- Progress is saved locally in the browser/Home Screen app.
- Use Export/Import to back up or move progress.
- The app works offline after the first successful visit.
- If you update files on GitHub, iOS may take a little while to refresh the cached PWA.
